GenGein Engine

GenGein is still currently an on-going experiment, but GenGein is an engine I wrote with C++ and OpenGL. It currently has the potential to produce complex game systems, accurate real-time physics calulations and modern graphics techniques/rendering. PhysVsPhysX and Checkers are both great examples of GenGein's potential.



Columns

Columns is a 'match 3' game that allowed me to explore different programming languages. Those including JavaScript (Cocos2D Framework), JQuery, HTML and CSS. This project lasted for about 5 weeks and is also compatible on mobile devices. Columns also stores highscores locally and has a built in live weather API.



Gravitas

The concept of Gravitas is simple. Alter the direction of gravity to explore the worlds in search for the destination, only hoping to avoid the turrets. It's written in C# Monogame with the physics engine 'Farseer Physics'. Gravitas is compatible on both PC and PS Vita [Development build] thanks to PlayStation Mobile.
